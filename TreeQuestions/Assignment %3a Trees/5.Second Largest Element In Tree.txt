Assignment Coding Problem
Problem Name: Second Largest Element In Tree
Problem Level: MEDIUM
Problem Description: #### Given a generic tree, find and return the node with second largest value in given tree. Return NULL if no node with required value is present.

##### Input format : 

    Elements in level order form separated by space (as per done in class). Order is - 

    Root_data, n (No_Of_Child_Of_Root), n children, and so on for every element 

##### Output format:
    Second Largest node data
 ##### Sample Input 1 : 
    10 3 20 30 40 2 40 50 0 0 0 0 

##### Sample Output 1 : 
    40
